    linkerd

    Ultralight service mesh for Kubernetes

    Linkerd is an ultralight, security-first service mesh for Kubernetes. Linkerd adds critical security, observability, and reliability features to your Kubernetes stack with no code change required.

    devops

    https://linkerd.io

    amd64 armhf arm64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install linkerd

    sudo apt install linkerd

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    linkerd=25.7.6*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      linkerd=25.7.6*
