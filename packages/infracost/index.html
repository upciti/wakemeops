    infracost

    Cloud cost estimates for Terraform in pull requests

    Infracost shows cloud cost estimates for Terraform. It lets DevOps, SRE and engineers see a cost breakdown and understand costs before making changes, either in the terminal or pull requests.

    devops

    https://www.infracost.io

    arm64 amd64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install infracost

    sudo apt install infracost

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    infracost=0.10.43*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      infracost=0.10.43*
