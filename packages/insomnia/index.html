    insomnia

    Collaborative API Client and Design Tool

    Insomnia is an open-source, cross-platform API client for GraphQL, REST, and gRPC.

    desktop

    https://insomnia.rest/

    amd64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install insomnia

    sudo apt install insomnia

    Blueprints

    Debian packages listed on this page are generated from op2deb YAML blueprints. Blueprints for insomnia are versioned here.

    Click here to see insomnia ops2deb blueprints
    https://github.com/upciti/wakemeops/blob/main/blueprints/desktop/insomnia/ops2deb.yml
    name: insomnia
matrix:
  versions:
    - 2023.3.0
    - 2023.4.0
    - 2023.5.8
homepage: https://insomnia.rest/
summary: collaborative API Client and Design Tool
description: |-
  Insomnia is an open-source, cross-platform API client for GraphQL, REST, and
  gRPC.
fetch: https://github.com/Kong/insomnia/releases/download/core%40{{version}}/Insomnia.Core-{{version}}.deb
install:
  - data:/
script:
  - grep "^Depends. " control/control >> {{debian}}/control
  - sed -i "s/dh_strip --no-ddebs/true/" {{debian}}/rules

    The blueprint fetch keyword contains a URL template pointing to insomnia releases. Downloaded files are locked in a lockfile versioned here.
