    lazydocker

    A lazier way to manage everything docker

    A simple terminal UI for both docker and docker-compose, written in Go with the gocui library.

    devops

    https://github.com/jesseduffield/lazydocker

    armhf arm64 amd64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install lazydocker

    sudo apt install lazydocker

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    lazydocker=0.24.4*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      lazydocker=0.24.4*

matrix:
  architectures:

summary: a lazier way to manage everything docker

  A simple terminal UI for both docker and docker-compose, written in Go with the
  gocui library.
fetch:
  url: https://github.com/jesseduffield/lazydocker/releases/download/v{{version}}/lazydocker_{{version}}_Linux_{{target}}.tar.gz

    amd64: x86_64
