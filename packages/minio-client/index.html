    minio-client

    Command-line interface for MinIO (mc)

    MinIO Client (mc) is a replacement for ls, cp, mkdir, diff and rsync commands for filesystems and object storage.

    devops

    https://min.io

    amd64 arm64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install minio-client

    sudo apt install minio-client

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    minio-client=2024-11-05T11-29-45Z*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      minio-client=2024-11-05T11-29-45Z*
