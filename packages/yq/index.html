    yq

    Portable command-line YAML processor

    Lightweight and portable command-line YAML processor. yq uses jq like syntax but works with yaml files as well as json. It doesn't yet support everything jq does - but it does support the most common operations and functions, and more is being added continuously.

    terminal

    https://github.com/mikefarah/yq

    amd64 armhf arm64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install yq

    sudo apt install yq

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    yq=4.48.1*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      yq=4.48.1*

    - |-
      set -eu
      if [ "$(dpkg --print-architecture)" = '{{arch}}' ]; then
        yq='{{src}}/usr/bin/yq'
        chmod 755 "$yq"
        mkdir -p {{src}}/usr/share/bash-completion/completions
        "$yq" completion bash > {{src}}/usr/share/bash-completion/completions/yq
        mkdir -p {{src}}/usr/share/zsh/vendor-completions
        "$yq" completion zsh > {{src}}/usr/share/zsh/vendor-completions/_yq
        mkdir -p {{src}}/usr/share/fish/vendor_completions.d
