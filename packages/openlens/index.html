    openlens

    Powerful IDE for kubernetes

    OpenLens (aka Lens Desktop Core) is the open source project on which Lens Desktop is built. It provides the full situational awareness for everything that runs in Kubernetes. It’s lowering the barrier of entry for people just getting started and radically improving productivity for people with more experience.

    Lens is built on open source and backed by a number of Kubernetes and cloud native ecosystem pioneers. With over 350,000 users and 16,000 stars on GitHub, it’s the largest and most advanced Kubernetes platform in the world.

    desktop

    https://github.com/MuhammedKalkan/OpenLens

    arm64 amd64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install openlens

    sudo apt install openlens

    Blueprints

    Debian packages listed on this page are generated from op2deb YAML blueprints. Blueprints for openlens are versioned here.

    Click here to see openlens ops2deb blueprints
    https://github.com/upciti/wakemeops/blob/main/blueprints/desktop/openlens/ops2deb.yml
    name: openlens
matrix:
  architectures:
    - amd64
    - arm64
  versions:
    - 6.4.15
    - 6.5.2-366
    - 6.5.2
homepage: https://github.com/MuhammedKalkan/OpenLens
summary: powerful IDE for kubernetes
description: |-
  OpenLens (aka Lens Desktop Core) is the open source project on which Lens
  Desktop is built.
  It provides the full situational awareness for everything that runs in
  Kubernetes. It’s lowering the barrier of entry for people just getting started
  and radically improving productivity for people with more experience.

  Lens is built on open source and backed by a number of Kubernetes and cloud
  native ecosystem pioneers. With over 350,000 users and 16,000 stars on GitHub,
  it’s the largest and most advanced Kubernetes platform in the world.
fetch: https://github.com/MuhammedKalkan/OpenLens/releases/download/v{{version}}/OpenLens-{{version}}.{{arch}}.deb
install:
  - data:/
script:
  - grep "^Depends. " control/control >> {{debian}}/control
  - sed -i "s/dh_strip --no-ddebs/true/" {{debian}}/rules

    The blueprint fetch keyword contains a URL template pointing to openlens releases. Downloaded files are locked in a lockfile versioned here.
